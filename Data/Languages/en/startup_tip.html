Welcome to Floating Sandbox!

Before you get started, we've got just a few tips to help you find your way:

  • To load a ship go to the File menu and select Load Ship
  • To change the view go to the Controls menu
  • To use tools go to the Tools menu and select a tool
  • To dim daylight use the Page Up/Page Down keys or use the Controls menu
  • To edit or create a ship go to the ShipBuilder menu
  • To get more help go to the Help menu and select Help

Enjoy!
