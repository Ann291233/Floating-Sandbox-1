Overview

This game is a realistic two-dimensional mass-spring network, which simulates physical bodies (such as ships) floating in water. You can punch holes in an object, slice it, apply forces to it, and smash it with bomb explosions. Once water starts rushing inside the object, it will start sinking and, if you're lucky enough, it will start breaking up!

Our philosophy in making this game is to simulate the real world as closely as possible. Each new release adds new ways in which particles interact among themselves and with the world that surrounds them; think of gravity, buoyancy, and wind, but also bouncing when smashing on the oceean floor, and feeling drag when moving fast underwater.

You can create your own physical objects by drawing images using colors that correspond to materials in the game's library. Each material has its own physical properties, such as mass, strength, stiffness, water permeability, sound properties, electrical properties, and so on. If you want, you can also apply a higher-resolution image to be used as a more realistic texture for the object. Once you load your object, watch it float and explore how it behaves under stress! Read the "How to Build a Ship" guide under the "Guides" directory for an explanation on how to build your own objects.

The game currently comes with a few example objects - mostly ships. You can find them under the "Ships" directory.

Tools

Once you load a ship, you can do quite a few things with it.
First of all, you can use the Smash tool to punch holes in it. The longer you keep the mouse button pressed, the larger the hole you can punch! Then there's the Slice tool, with which you can slice your ship.
The Grab tool and the Swirl tool apply radial and tangent forces, respectively. Also with these tools, the longer you keep the mouse button pressed, the stronger the forces become. You can invert the direction of the forces with the key.
The Pin tool allows you to "freeze" a point of the ship: once the point is frozen, its velocity becomes zero and its position never changes. It's best to increase the overall strength of the ship via the Settings Dialog when you use this tool, or else the frozen point will quickly break under the enormous stress brought by neighboring points. A pin can be removed by applying the tool to the same point again.
With the Move tool you can move (or rotate, if pressing the key) parts of a ship, while the Move All tool allows you to move (or rotate) an entire ship.
The Flood tool allows you to inject water into a ship, or to extract water from a ship when pressing the key.
After a ship has sunk to the bottom, its structure tends to rot; you can use the Scrub tool to clean the dirt away.

Finally, for the passive-aggressive types, ultimate destruction can be achieved lazily with bombs.
Timer Bombs explode automatically a few seconds after having been placed, while Remote Controlled Bombs and Anti-Matter Bombs await for you to detonate them with the and keys, respectively. Impact Bombs just explode on...impact.

If you feel remorse for the destruction you have inflicted, you may try the Repair Structure tool on the broken edges of a ship, and if you are skillful enough, you'll be able to restore the structure of the ship to its pristine condition.

View Control

You can zoom the view with the and keys or with your mouse's wheel, while you can pan the view using the keys. If you get lost, the key resets the view to its default coordinates.
Darkness and light can be adjusted with the and keys.
